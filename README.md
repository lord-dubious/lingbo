# Lingbo - Learn Igbo The Natural Way 🌿

**Lingbo** is an interactive, dual-interface web application designed to make learning the Igbo language accessible, fun, and engaging for all ages. Whether you are an adult looking to reconnect with your roots or a child taking your first steps into the language, Lingbo provides a tailored experience just for you.

Powered by **Google's Gemini API**, Lingbo integrates advanced AI features like a real-time tutor, speech synthesis, and pronunciation analysis to accelerate your learning journey.

---

## ✨ Key Features

### 👥 Dual Learning Profiles
-   **Adult Mode**: A structured, curriculum-based approach focusing on grammar, vocabulary, and cultural insights.
-   **Kids Corner**: A colorful, gamified interface with interactive stories, games, and simplified lessons.

### 🤖 AI-Powered Learning (Gemini)
-   **Tutor Chat (Chike)**: Practice conversation with an AI tutor who understands context and provides instant feedback.
-   **Pronunciation Coach**: Speak into the microphone and get real-time analysis of your pronunciation.
-   **Speech Synthesis**: Hear any word or phrase spoken in natural-sounding Igbo.

### 🎮 Interactive Games (Kids Corner)
-   **Memory Match**: Find matching pairs of words and images with 3D card animations.
-   **Sentence Builder**: Drag and drop blocks to construct correct Igbo sentences.
-   **Speed Tap**: Race against the clock to identify the correct image for a given word.

### 📚 Rich Resources
-   **Library**: Access digital storybooks and workbooks.
-   **Video Lessons**: Curated video content for visual learners.
-   **Cultural Insights**: Learn about Igbo traditions, food, and daily life.

---

## 🛠️ Tech Stack

-   **Frontend**: React 19, TypeScript, Vite
-   **Styling**: Tailwind CSS, Lucide React Icons
-   **AI Integration**: Google GenAI SDK (Gemini 2.5 Flash)
-   **Routing**: React Router DOM

## 🚀 Getting Started

Follow these steps to run Lingbo locally on your machine.

### Prerequisites
-   **Node.js** (v18 or higher)
-   **npm** (comes with Node.js)

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/yourusername/lingbo.git
    cd lingbo
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Set up Environment Variables**
    Create a `.env.local` file in the root directory and add your Google Gemini API key:
    ```env
    GEMINI_API_KEY=your_api_key_here
    ```

4.  **Run the development server**
    ```bash
    npm run dev
    ```

5.  **Open your browser**
    Navigate to `http://localhost:5173` to start learning!
